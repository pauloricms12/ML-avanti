1. Machine Learning é, como o próprio nome sugere, aprendizado de máquina, ou seja, tornar a máquina um objeto "inteligente", a partir de treinos que a faz aprender padrões e, a partir destes, realizar ações, como extrair dados (por exemplo, um modelo em câmera de segurança para detectar placas de carro), detectar coisas (por exemplo, um modelo para detectar expressões faciais), etc.
2. Conjunto de treinamento é um Dataset para que o modelo de Machine Learning seja treinado e aprenda os padrões desejados. Já o conjunto de validação, é um conjunto de dados que serão utilizados para validar o modelo treinado. A validação pode ocorrer de várias formas, mas a mais comum é a acurácia, que mede o quão acertivo está o modelo. E, por último, o conjunto de teste é o conjunto para fazer predções do modelo já treinado e validado.
3. Eu faria preprocessamento dos dados (imputação, o mais comum), de acordo com o tipo de dados que está sendo trabalhado. Caso os dados sejam variáveis do tipo quantitativa contínuas, eu utilizaria imputação pela média. Caso sejam discretas, imputação com base na média. E assim por diante.
4. A matriz de confusão avalia o modelo a partir de falsos positivos, falsos negativos, verdadeiros positivos e verdadeiros negativos. E a partir disso, ajuda a melhorar o modelo.
5. Tenho interesse na área de saúde e acessibilidade.
